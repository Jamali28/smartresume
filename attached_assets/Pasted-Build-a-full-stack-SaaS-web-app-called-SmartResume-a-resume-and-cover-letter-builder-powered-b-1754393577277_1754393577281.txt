Build a full-stack SaaS web app called **SmartResume** — a resume and cover letter builder powered by AI (Google Gemini Pro).

✅ Project Overview:
A professional resume generator with:
- Multi-step resume form
- AI-powered resume rewriting and cover letter generation
- Tone selector (formal, friendly, neutral)
- PDF export (with watermark for free users)
- User authentication and saved resume history
- Optional Stripe payment integration

---

👨‍💻 Stack:

Frontend:
- React + TypeScript
- Tailwind CSS
- Multi-step resume builder form with preview
- Dashboard for saved resumes
- Responsive and clean UI

Backend:
- Node.js with Express or Next.js (API Routes)
- Supabase (for database + authentication)
- Puppeteer or react-pdf for PDF export

AI Integration:
- Use **Google Gemini Pro** via `@google/generative-ai`
- Pass environment variable `GOOGLE_API_KEY`
- Features:
  - Rewrite resume based on job description
  - Generate personalized cover letter
  - Calculate match score (resume vs job)

Payment (Optional for now):
- Integrate **Stripe** (freemium model)
- Use test keys now
- Free tier: 1 resume, watermark, limited templates
- Paid tier: unlimited resumes, no watermark, all templates
- Use `/api/create-checkout-session` + webhook for billing
- Hide paid options for now if not integrating Stripe immediately

---

🔐 User Flow:

1. Sign up / log in (Supabase Auth)
2. Fill out resume builder form (name, education, experience)
3. Paste job description
4. Select tone (dropdown)
5. Click “Generate with AI”
6. Show optimized resume + cover letter
7. Allow PDF export
8. Optional: Upgrade to premium via Stripe

---

🗃️ Database Schema (Supabase):

- `users`: id, email, name
- `resumes`: id, user_id, data (JSON), created_at
- `cover_letters`: id, resume_id, text, created_at
- `subscriptions`: user_id, stripe_customer_id, is_active
